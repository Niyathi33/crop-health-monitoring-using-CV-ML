# crop-health-monitoring-using-CV-ML
# Automated Crop Health Monitoring using Computer Vision and Machine Learning

## Overview
This project focuses on detecting crop diseases using computer vision and machine learning techniques. 
The system analyzes plant leaf images and predicts whether the crop is healthy or affected by disease.

This project was developed as part of a Computer Vision / Machine Learning academic project.



## Objectives

- Detect crop health condition using image data
- Apply image preprocessing using OpenCV
- Use deep learning / transfer learning for classification
- Evaluate model performance using confusion matrix and accuracy metrics
- Test model on unseen images



## Technologies Used

- Python
- TensorFlow / Keras
- OpenCV
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn
- Pillow



## Project Files

- `crop_health_monitoring.ipynb` → Main notebook
- `project_report.docx` → Project report
- `requirements.txt` → Required libraries
- `data/` → Dataset folder (not included)
- `results/` → Sample output images



## Dataset

The dataset used in this project is large and cannot be uploaded to GitHub.

To run the project:

1. Create a folder named `data`
2. Place the dataset inside the `data/` folder
3. Update dataset path in the notebook if needed

Example:
data/
train/
test/

## Results

The model is able to classify crop images into healthy and diseased categories.
